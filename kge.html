Knowledge Graph Engineering Project

Tourism and Waste Management in Trentino

Project Overview

This project aims to create a Knowledge Graph (KG) that integrates data on tourism and waste management in the Province of Trento. The graph provides valuable insights for stakeholders such as tourists, facility owners, and policymakers to promote sustainable tourism and effective waste management practices.

Purpose and Scope

The Knowledge Graph covers key domains such as waste production, tourist attractions, and municipal waste regulations. Data sources include OpenStreetMap, Dolomiti Ambiente, and the Italian Institute for Environmental Protection and Research (ISPRA).

Core Components

  • Entity-Relationship (ER) Model
  • Teleontology using the kTelos methodology
  • SPARQL Querying for Competency Questions

Evaluation and Results

The Knowledge Graph was evaluated for purpose satisfaction and reusability. Key metrics included entity and property connectivity, with strong performance across both dimensions.

Example SPARQL Query

SELECT ?municipality ?totalWaste 
WHERE {
  ?wasteProduction ex:generatedBy ?municipality .
  ?wasteProduction ex:wasteType "Plastic" .
  ?wasteProduction ex:year "2022" .
  ?wasteProduction ex:quantity ?totalWaste .
}

This query retrieves the total amount of plastic waste generated in each municipality in 2022.
